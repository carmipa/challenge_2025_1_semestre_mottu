## <a name="sobre-o-projeto"></a>📑 **Sobre o Projeto**

### 💡 Gestão de Frota e Mapeamento Inteligente
O projeto visa aprimorar a gestão de frotas de motos em pátios de múltiplas filiais, utilizando **Visão Computacional** e **IoT** para monitoramento em tempo real.

### 🛠️ Funcionalidades
- **Identificação precisa** das motos por visão computacional  
- **Modelo digital** do pátio em tempo real  
- **Interface intuitiva** para operadores

### 🚀 Escalabilidade
A solução é pensada para +100 filiais no Brasil e México, com diferentes layouts de pátio.

## <a name="problemas-identificados"></a>🚩 **Problemas Identificados**

1. Dificuldade no mapeamento preciso das motos (eficiência operacional).  
2. Falta de visibilidade em tempo real, tornando a operação menos ágil.
